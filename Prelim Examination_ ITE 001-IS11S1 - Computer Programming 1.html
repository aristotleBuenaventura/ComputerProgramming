    Preliminary Examination

     

    This activity supports the attainment of the following Intended Learning Outcome/s:

    1. Summarize the historical development of computer programming;

    2. Examine the programming features and process of C++ programming language;

    3. Analyze the declaration of variables , identifiers, constant and data types;

    4. Evaluate the different operators appropriately;

    5. Create a basic program using C++ programming language;

    6. Compare the different types of Control Structures in problem solving; and
    7. Create a program using Control structures in C++ Programming Language.

    Instruction:  Read each question carefully. Click the option that best describe your answer.
